# Politique de confidentialité — Crypto Net Profit

**Version 1.0 — mise à jour le 23 juillet 2026**

## En bref

Vos données financières restent sur votre appareil. Crypto Net Profit ne les
transmet, ne les vend et ne les partage avec personne.

## Données traitées

Les montants, quantités, frais, libellés et résultats sont traités uniquement
dans l’extension. Un scénario n’est conservé que lorsque vous choisissez
explicitement de l’enregistrer.

## Stockage et suppression

Jusqu’à dix scénarios récents peuvent être enregistrés dans
`chrome.storage.local`. Ils restent dans le profil Chrome local et peuvent être
supprimés individuellement ou tous ensemble depuis l’écran Historique.

## Réseau, tiers et suivi

L’extension n’effectue aucune requête réseau et n’utilise aucun outil d’analyse,
publicité, cookie, compte, portefeuille, clé API ou service tiers. Elle ne
demande aucun accès aux pages que vous consultez.

## Portée du service

Les résultats sont des simulations mathématiques et ne constituent pas un
conseil financier. Aucune transaction n’est exécutée.

## Contact

Pour toute question relative à cette politique, utilisez l’adresse d’assistance
indiquée sur la fiche officielle de l’extension dans le Chrome Web Store.
